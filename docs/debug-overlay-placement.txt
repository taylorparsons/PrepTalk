ANIMATION SEQUENCE (Quality Downgrade Example)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Network degrades (detected by preflight-audio.js)
   ↓

2. Quality indicator changes:
   - Text: "MEDIUM" → "LOW" (instant)
   - Dot: Yellow → Red (instant)
   - Bar: 60% → 30% (500ms smooth transition)
   ↓

3. Stats update:
   - Sample Rate: 24 kHz → 16 kHz
   - Frame Size: 40 ms → 60 ms
   - Network: 4g → 3g
   - Latency: 85 ms → 250 ms
   - Bandwidth: 8.2 Mbps → 1.2 Mbps
   ↓

4. Toast appears:
   - Fade in from bottom (300ms)
   - Yellow border (warning)
   - ⬇️ icon + message
   ↓

5. Toast auto-dismisses:
   - Wait 4 seconds
   - Fade out (300ms)
